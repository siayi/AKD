Memulai Aplikasi Akuntansi Pemerintah Desa Berbasis Akrual


Aplikasi ini menggunakan Ms. Excel yang dilengkapi dengan fasilitas macro. Fasilitas macro ini bukan default dari Ms. Excel, sehingga pada saat membuka aplikasi ini untuk pertama kali, kadang-kadang muncul masalah yaitu tombol-tombol tidak bisa di-klik. Berikut ini adalah solusinya.

Pada komputer yang baru membuka file aplikasi ini, biasanya muncul peringatan Security Warning: Automatic update has been disabled. Pada sebelah kanan peringatan tersebut ada kata-kata Enable Content. Klik Enable Content tersebut, dan aplikasi siap dijalankan. 

Jika ternyata masih belum running juga, maka aktifkan menu Developer, dengan cara klik File, lalu pilih Options, selanjutnya klik Customize Ribbon, dan aktifkan menu Developer. Kalau menu Developer sudah muncul, maka klik menu Developer tersebut, lalu pilih Macro Security. Pada menu ini tinggal aktifkan Enable All Macros, dan terakhir klik OK.

File -> Options  -> Customize Ribbon -> Aktifkan Developer ->Macro Security -> Enable all macros
